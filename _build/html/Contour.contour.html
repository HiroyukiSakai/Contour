Contour.contour.set_metrics.hausdorff_distance(a, b)[source]¶

Calculate the Hausdorff distance [1] between two sets.

Parameters:
  • a (ndarray.) – Array containing the coordinates of N points in an M dimensional space.
  • b (ndarray.) – Array containing the coordinates of N points in an M dimensional space.
Returns:

float – The Hausdorff distance between the sets represented by a and b using Euclidian distance to calculate the distance between members of the sets.
[1]http://en.wikipedia.org/wiki/Hausdorff_distance
Contour.contour.set_metrics.hausdorff_distance_region(a, b)[source]¶

Calculate the Hausdorff distance [2] between two binary images.

Parameters:
  • a (ndarray, dtype=bool.) – Array where True represents a point that is included in a set of points. Both arrays must have the same shape.
  • b (ndarray, dtype=bool.) – Array where True represents a point that is included in a set of points. Both arrays must have the same shape.
Returns:

float – The Hausdorff distance between the sets represented by a and b using Euclidian distance to calculate the distance between members of the sets.
[2]http://en.wikipedia.org/wiki/Hausdorff_distance
